## 🏥 Descripción

Sistema de gestión oftalmológica desarrollado para clínicas, consultorios y hospitales especializados en salud visual.  
Permite administrar pacientes, historiales clínicos, diagnósticos visuales, recetas oftálmicas y procesos administrativos de manera eficiente.

---

## 🚀 Características Principales

### 👨‍⚕️ Gestión de Pacientes
- Registro de pacientes  
- Historial clínico oftalmológico  
- Control de consultas  
- Seguimiento de tratamientos  

### 👁️ Área Oftalmológica
- Evaluación de agudeza visual  
- Registro de refracción (miopía, hipermetropía, astigmatismo)  
- Prescripción de lentes  
- Diagnóstico de enfermedades oculares  
- Registro de presión intraocular (PIO)  

### 🧪 Estudios y Diagnóstico
- Exámenes de la vista  
- Resultados clínicos  
- Reportes médicos  
- Historial de estudios oftalmológicos  

### 💊 Tratamientos y Recetas
- Recetas médicas  
- Prescripción de lentes y medicamentos  
- Seguimiento de tratamientos  

### 🏥 Administración
- Gestión de doctores y personal  
- Agenda de citas  
- Control de pagos  
- Reportes administrativos
